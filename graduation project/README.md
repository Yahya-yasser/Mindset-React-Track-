# Movie Explorer

A simple React application to explore movies, view details, and manage a favorites list.

## Features
- **Home**: View popular movies and search for specific titles.
- **Details**: Click on a movie to see detailed information including plot, cast, and rating.
- **Favorites**: Add movies to your favorites list (persisted in local storage).
- **Responsive Design**: Works on desktop and mobile.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5174 in your browser.

## Technologies
- React
- React Router
- Context API (for state management)
- CSS (Vanilla)
