# Todo Vite React

Minimal Vite + React todo app with add / toggle done / delete + localStorage.

Run (Windows PowerShell):

```powershell
npm install
npm run dev
```

Open the URL printed by `vite` (usually `http://localhost:5173`).

Files of interest:
- `src/App.jsx` — main app logic
- `src/components/TodoInput.jsx` — input + add button
- `src/components/TodoList.jsx` — list, toggle, delete
- `src/styles.css` — styles to match dark design
